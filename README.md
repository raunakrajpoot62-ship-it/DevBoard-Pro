# DevBoard-Pro
A modern productivity dashboard with task management, notes, Pomodoro timer, habit tracking, bookmarks, dark mode, local storage support, and a realistic responsive UI built using HTML, CSS, and JavaScript.

---

## ✨ Features

- ✅ Task Manager
- 📝 Notes System
- ⏱️ Pomodoro Timer
- 🔁 Habit Tracker
- 🔖 Bookmark Manager
- 🌙 Dark / Light Theme
- 💾 Offline localStorage Support
- 📱 Responsive Mobile UI
- 📂 Import / Export Backup

---

## 📸 Preview

DevBoard is a clean and modern productivity dashboard designed for students, developers, and creators.

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage API
